# gatsby-source-nhl

This source plugin for Gatsby will pull data from the NHL and make it available in GraphQL queries.

## Installation

```sh
# Install the plugin
yarn add gatsby-source-nhl
```

In `gatsby-config.js`:

```js
module.exports = {
  plugins: [
    {
      resolve: "gatsby-source-nhl",
      options: {},
    },
  ],
}
```
